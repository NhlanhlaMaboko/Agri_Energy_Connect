#  Agri_Energy_Connect

Agri_Energy_Connect is a web application designed to bridge the gap between **agriculture and energy management**. It provides farmers, cooperatives, and stakeholders with tools to monitor, manage, and optimize resources in a secure, cloud-ready environment.

---

##  Features
-  **Data Management**: Store and manage agricultural and energy data securely.
- **Secure Architecture**: Built with ASP.NET Core and follows best practices for authentication and authorization.
-  **Responsive UI**: Modern front-end with Razor Views and static assets in `wwwroot`.
-  **Database Integration**: Entity Framework Core with migrations for SQL Server.
-  **Cloud Ready**: Configurable for Azure SQL Database and Blob Storage.

---

##  Tech Stack
- **Backend**: ASP.NET Core (C#)
- **Frontend**: Razor Views, HTML, CSS, JavaScript
- **Database**: SQL Server (EF Core migrations included)
- **Cloud**: Azure SQL + Blob Storage (optional)
- **UI/UX**: Figma prototypes for design

---

## Project Structure
- `Controllers/` – MVC controllers
- `Models/` – Domain models
- `ViewModels/` – Data transfer and view models
- `Views/` – Razor views
- `Data/` – EF Core DbContext
- `Migrations/` – Database migrations
- `wwwroot/` – Static assets (CSS, JS, images)
